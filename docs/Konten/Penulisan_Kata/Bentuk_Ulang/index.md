# Bentuk Ulang

Bentuk ulang ditulis dengan menggunakan tanda hubung \``-`\` di
antara unsur-unsurnya.

???+ tip "Misal"
    <table>
      <tr>
        <td>anak-anak</td>
        <td>biri-biri</td>
    </tr>
      <tr>
        <td>buku-buku</td>
        <td>cumi-cumi</td>
      </tr>
      <tr>
        <td>hati-hati</td>
        <td>kupu-kupu</td>
      </tr>
      <tr>
        <td>kura-kura</td>
        <td>kuda-kuda</td>
      </tr>
      <tr>
        <td>lauk-pauk</td>
        <td>berjalan-jalan</td>
      </tr>
      <tr>
        <td>mondar-mandir</td>
        <td>mencari-cari</td>
     </tr>
      <tr>

        <td>terus-menerus</td>
      </tr>
      <tr>

      </tr>

      </tr>
    </table>
    
???+ note "Catatan"
    Bentuk ulang gabungan kata ditulis dengan mengulang unsur pertama.

    !!! tip "Misal"
        surat kabar  
        → surat-surat kabar  
        kapal barang  
        → kapal-kapal barang  
        rak buku  
        → rak-rak buku  
        kereta api cepat  
        → kereta-kereta api cepat
