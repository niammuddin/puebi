# Bentuk Ulang

!!! summary " "
    Bentuk ulang ditulis dengan menggunakan tanda hubung <span class="penanda">-</span> di antara unsur-unsurnya.

    !!! summary " "
        Bentuk ulang gabungan kata ditulis dengan mengulang unsur pertama.
